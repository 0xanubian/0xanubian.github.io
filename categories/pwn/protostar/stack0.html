#include <stdlib.h>
#include <unistd.h>
#include <stdio.h>

int main(int argc, char **argv)
{
  volatile int modified;
  char buffer[64];

  modified = 0;
  gets(buffer);

  if(modified != 0) {
      printf("you have changed the 'modified' variable\n");
  } else {
      printf("Try again?\n");
  }
}

  1. ebp is pushed
  2. esp is moved into ebp
  3. esp is and with 0xfffffff0. It will convert the last 4 bit of esp to 0.
  4. esp - 0x60. Now the stack pointer points lower than base pointer
  5. This creates a stack frame where variables can be stored
  6. 0x0 is moved to the memory location at esp+0x5c
  7. we suspect that this is our variable that we will have to modify
  8. gets is called
  9. what is stored in esp+0x5c is moved to eax
  10. test instruction is done on eax. test set the zero flag when result of and operation is zero.
  11. if zero flag is turned on we jump to 0x08048427 and print ““you have changed the modified variable”
  12. otherwise try again is printed.
  13. For pwning this challenge we have to overflow the stack and change the value at esp+0x5c.
